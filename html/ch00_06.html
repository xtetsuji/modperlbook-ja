0.6. Apache and Perl Versions

0.6. ApacheとPerlのバージョン

We have used mod_perl 1.26 and Apache 1.3.24 in most of the examples in this book. You should be able to reproduce all the presented examples with these or later versions of mod_perl and Apache.

我々はこの書籍中の多くの例の中で mod_perl 1.26 と Apache 1.3.24 を 使っていきます。これらもしくは最新のバージョンの mod_perl と Apache で 与えられた例の数々を全て模倣することができるようになるべきです。 もちろんこの記述はApache1.3/mod_perl1系の話です。 Apache2.x/mod_perl2系は全く別構文を持つのでそれらは別と考えてください

We have tested all the examples with Perl 5.6.1. However, most of the examples should work the same under all Perl versions between 5.005_03 and 5.8.0.

我々はPerl5.6.1にて全ての例をテストしています。 しかしながら例の多くは5.005_03から5.8.0の間の全てのPerl上で同様に 動作することでしょう。 2013年現在、Perl5.8.1よりも前のバージョンは モジュール保守の対象から完全に外れています。Perl5.8.0は文字コード周りの バグがあり、Perl5.8系からも見放された存在であることに注意してください。 また2013年現在Perl5.18が最新版として存在し、Perl5.14以前は保守の対象外と なっていることにも留意してください。しかし、2013年5月現在、Perl5.16で mod_perl2が動作するよう保守が続けられていますがmod_perl1については 過度な期待はできないと思われます。

At the time of this writing, Apache 2.0 is very young and mod_perl 2.0 is still in development. See Part V for information on mod_perl 2.0. While much of this book should apply to both mod_perl 1.x and mod_perl 2.0, the code has been tested only on mod_perl 1.26.

これを書いている時点で、Apache 2.0 が非常に若く、 依然として mod_perl 2.0 が開発されています。 mod_perl 2.0 の情報については パート5を見てください。 この書籍「Practical mod_perl」が書かれた2003年現在はそのような状況でしたが、この書籍がmod_perl 2として載せた事例は、現在では「mod_perl 1.99」と呼ばれる、2013年現在のmod_perl 2と古いmod_perl 1のキメラのような奇怪な存在として存在します。そのようなmod_perl 1.99を同梱したLinuxディストリビューションも存在するので使用実例も存在しますが、現在特段の理由がない限り今現在「mod_perl 1.99」を使うことは強く忌避されることでしょう。この訳注では、mod_perl 2を称した「mod_perl 1.99」について、本当のmod_perl 2の解説を出来る限り入れていきます。 この書籍の多くの事柄がmod_perl 1.x や mod_perl 2.0 の両方に適用することが できるである間ずっと、そのコードは mod_perl 1.26 上のみでテストされます。 While を「間ずっと」と訳していますが、 これは誤訳かと思います。上手い訳が思いつきませんでした
